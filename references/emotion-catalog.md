# 情绪类型库

## 基础情绪（7 种）

| 情绪 | 强度范围 | 常见表现 |
|------|---------|---------|
| **喜** | 微笑→狂喜 | 笑声、分享欲、轻快举止 |
| **怒** | 烦躁→暴怒 | 沉默or爆发、攻击性 |
| **哀** | 淡淡的忧伤→崩溃 | 流泪、沉默、退缩 |
| **惧** | 不安→恐慌 | 逃避、僵硬、过度警觉 |
| **爱** | 好感→痴迷 | 关注、牺牲、靠近 |
| **恶** | 厌恶→仇恨 | 排斥、攻击、冷漠 |
| **欲** | 渴望→贪婪 | 追逐、不安于现状 |

## 复杂情绪频谱

这些是基础情绪的变体和组合，编剧常用：

### 孤独系
```
轻度 → 中度 → 重度
独处自洽 → 无人理解 → 存在性孤独
「一个人也挺好」→「说什么都没人懂」→「我不属于这个世界」
```

| 子类型 | 强度 | 典型场景 |
|--------|------|---------|
| 社交孤独 | 中 | 人群中感到格格不入 |
| 情感孤独 | 中-高 | 有伴侣但心不在一起 |
| 存在性孤独 | 高 | 对生命意义的虚无感 |
| 被抛弃感 | 高 | 被重要的人/群体排斥 |

### 怀旧系
```
温暖怀旧 → 苦涩怀旧 → 创伤性闪回
「那时候真好」→ 「再也回不去了」→ 「不想再想起」
```

| 子类型 | 强度 | 触发物 |
|--------|------|--------|
| 温暖怀旧 | 轻-中 | 老歌、旧照片、熟悉的气味 |
| 遗憾怀旧 | 中-高 | 错过的人、未选的路 |
| 创伤闪回 | 高 | 特定场景触发，伴随生理反应 |

### 荒诞系
```
黑色幽默 → 存在荒诞 → 虚无
「这也太搞笑了」→ 「一切都没有意义」→ 「那又怎样」
```

| 子类型 | 说明 |
|--------|------|
| 情境荒诞 | 正常人在不正常的世界 |
| 逻辑荒诞 | 事情的发展毫无因果 |
| 官僚荒诞 | 系统对个体的碾压（卡夫卡式） |
| 存在荒诞 | 努力和结果之间没有关系 |

### 成长系
```
不安 → 挣扎 → 突破 → 释然
「我行吗」→ 「我做不到」→ 「我再试一次」→ 「我做到了」
```

### 遗憾系
```
轻微遗憾 → 深切悔恨 → 无法挽回
「当时要是...」→ 「我毁了一切」→ 「已成定局」
```

### 矛盾感
```
爱恨交织 | 悲喜交加 | 恐惧又期待 | 厌恶又同情
```

## 情绪组合矩阵

两个对立情绪叠加产生的戏剧张力：

|   | 喜 | 怒 | 哀 | 惧 | 爱 | 恶 |
|---|---|---|---|---|---|---|
| **喜** | - | 狞笑 | 强颜欢笑 | 劫后余生 | 甜蜜 | 嘲讽 |
| **怒** | - | - | 悲愤 | 威慑 | 嫉妒 | 仇恨 |
| **哀** | - | - | - | 绝望 | 痴情 | 厌恶自己 |
| **惧** | - | - | - | - | 占有欲 | 避之不及 |
| **爱** | - | - | - | - | - | 可怜之人 |
| **恶** | - | - | - | - | - | - |

**使用说明**：好的情绪设计不是单一的，而是「笑着哭」「愤怒中的无力」「恐惧中的期待」。矩阵帮助你找到这种张力。

## 情绪强度标注规范

```
1-2：几乎察觉不到
3-4：隐约有感觉
5-6：明显但不影响行为
7-8：强烈，影响决策
9-10：压倒性的，完全支配行为
```

场景标注示例：
- 「他站在地铁站，看着自动售票机」→ 孤独：7，怀旧：5
- 「她翻开旧相册，突然哭了」→ 怀旧：8，遗憾：6，释放：3
