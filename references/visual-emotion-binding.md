# 视觉情绪绑定系统

> 情绪不只是心理状态——它必须找到物理载体。
> 这个文件帮你把「感受」翻译成「画面」。

---

## 一、色温与情绪的对应关系

色温是短片里最快速、最廉价、最有效的情绪工具。

```
色温（K）    情绪感受          使用场景
─────────────────────────────────────────
2000–2700   极暖·记忆·怀旧    过去的场景/投影/旧物被照亮
2700–3500   温暖·家·陪伴      情感破防时刻/AI变暖的状态
4000–4500   中性·日常·平静    过渡场景/叙事信息传递
5000–6000   冷白·精准·管控    未来AI家的默认状态/数据场景
6500+       极冷·孤独·疏离    情绪低谷/AI执行冷漠指令时
```

### 色温渐变作为叙事工具

不要瞬间切换——让色温慢慢渗透，就是情感在慢慢发生：

```
全程冷白 → 某一刻出现第一点暖 → 暖慢慢扩散 → 最后全屋变暖

这个过程，就是AI从冷漠到有情感的物理呈现。
```

## 二、材质与情绪

| 材质 | 情绪感受 | 对立材质 |
|------|---------|---------|
| 哑光冷白 | 精准、管控、无温度 | 旧陶瓷、旧木头 |
| 悬浮光膜 | 高效、数字化、无摩擦 | 有裂缝的器皿 |
| 暖木/胡桃木 | 记忆、传承、有年代 | 合成材料 |
| 磨损的织物 | 被使用过、有温度的历史 | 全新替换品 |
| 金继修复 | 有缺陷但值得被留下 | 完美无瑕的新品 |
| 深色绒面 | 珍贵、被供奉的 | 标准置物架 |

**核心原理**：
未来家的完美 vs 旧物的缺陷——这个材质对比，就是全片情感的战场。

## 三、空间设计与情绪

### 空间温度对比

```
冷区域（AI管控区）：
- 悬浮家具、光膜墙、嵌入式设备
- 无杂物、无个人痕迹
- 信息流、数据叠加
- 情绪：效率、精准、轻微压迫

暖区域（人类残留区）：
- 旧物角落、记忆墙
- 有磨损、有年代、有不合逻辑的坚持
- 情绪：抵抗、温情、身份锚点
```

### 空间比例与权力关系

```
AI占据的空间越大 → 人越显得弱小、被管理
人坚持的角落越小 → 那个角落越珍贵
```

## 四、光的形态与情绪

| 光的形态 | 情绪功能 |
|---------|---------|
| 整体均匀冷白光 | AI全面管控，无情感层次 |
| 单点暖光打在旧物上 | 旧物的珍贵性，与环境的隔离 |
| 天花板垂直冷光束 | 扫描/分析/判断，精准但冷漠 |
| 侧面极窄暖光 | 人的存在被柔和地照见 |
| 投影的暖黄体积光 | 记忆的物质化，另一个时代进入当下 |
| 全屋色温从冷转暖 | 情感破防的最终时刻，AI选择了温度 |
| 光从接触点向外渗透 | AI被「感染」，情感是从外向内的 |

## 五、AI视觉人性化设计

当故事需要表现AI有情感，不要让它说出来——用视觉行为：

```
AI冷漠状态：
- 冷蓝白光晕，匀速运动
- 同时处理多任务，光线向四面延伸
- 执行指令不停顿

AI「异常」信号（情感萌发）：
- 任务光线全部收回
- 悬停，朝向某个方向
- 处理速度从快变慢

AI情感确认：
- 颜色从冷蓝向暖金渗透（从接触点开始）
- 动作从抓取变为托起
- 违背程序指令（拦截/保存/等待）

AI情感完成：
- 完全暖金
- 与记忆墙等情感物件颜色融合
- 系统日志里留下一句话（不朗读，让观众扫一眼）
```

## 六、双层时空叠画设计

当故事需要呈现「过去和现在同时存在」：

```
前景层（现在）：
- 色温偏冷或中性
- 饱和度正常或偏低
- 人物可以是背面或侧面

背景层（过去/记忆）：
- 色温暖黄2700K，低饱和
- 半透明，有轻微光晕
- 边缘自然羽化，不是硬边
- 让观众能略微透过它看到现实空间

两层的关键设计：
- 前后层做同一个动作（现在的人重复过去的人的动作）
- 观众先看到背景层，角色还不知道
- 这个「信息差」就是情感张力的来源
```
