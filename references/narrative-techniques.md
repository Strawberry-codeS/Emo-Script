# 叙事手法库

叙事手法 = 讲故事的工具箱。每个场景可以标注用了什么手法。

---

## 一、时间操控

| 手法 | 说明 | 效果 |
|------|------|------|
| **倒叙** | 从结局或中间开始讲 | 营造悬念、改变认知 |
| **预叙** | 提前透露未来 | 制造宿命感/无力感 |
| **平行蒙太奇** | 两条线交替出现 | 对比、呼应、紧张感 |
| **交叉蒙太奇** | 两线越剪越快 | 最后时刻救援的张力 |
| **慢镜头** | 时间拉长 | 强调关键时刻、情感放大 |
| **跳切** | 省略过程直接跳 | 节奏快、现代感 |
| **时间循环** | 重复同一天/同一事件 | 困境感、成长感 |

---

## 二、视角操控

| 手法 | 说明 | 效果 |
|------|------|------|
| **限制视角** | 观众只知主角所知 | 代入感、悬疑 |
| **全知视角** | 观众比角色知道更多 | 戏剧反讽（替角色着急） |
| **不可靠叙述者** | 叙述者说谎/记错 | 反转、怀疑一切 |
| **多视角** | 同一事件多人叙述 | 《罗生门》效应 |
| **第三人称冷叙述** | 不带情感的旁白 | 荒诞感、距离感 |

---

## 三、情绪操控

| 手法 | 说明 | 效果 |
|------|------|------|
| **留白** | 不说、不展示关键信息 | 想象空间、余韵 |
| **反衬** | 用相反情绪烘托 | 以乐写哀、以静写动 |
| **重复** | 同一画面/台词反复出现 | 主题强化、情感累积 |
| **升格** | 从具体事件升到普遍意义 | 哲学感、格局打开 |
| **降格** | 从宏大叙事落到微小细节 | 接地气、情感落地 |
| **麦高芬** | 推动情节但本身不重要的东西 | 驱动故事，聚焦人物 |
| **契诃夫之枪** | 第一幕出现的东西第三幕必须用 | 前后呼应、严密感 |

---

## 四、结构操控

| 手法 | 说明 | 效果 |
|------|------|------|
| **反转** | 观众以为A，其实是B | 惊喜、重新理解前面 |
| **嵌套叙事** | 故事里讲故事 | 层次感、元叙事 |
| **冰山理论** | 只写1/8，7/8在水下 | 克制、深度感 |
| **环形结构** | 首尾呼应 | 宿命感、完成感 |
| **非线性** | 打乱时间顺序 | 拼图感、主动理解 |

---

## 五、场景内部手法

| 手法 | 说明 | 什么时候用 |
|------|------|-----------|
| **潜台词** | 角色说不等于角色想 | 每场都该有 |
| **动作即性格** | 不靠台词靠行为塑造 | 展示 > 告诉 |
| **环境即情绪** | 场景渲染人物内心 | 情绪高潮场景 |
| **静默的力量** | 关键时刻不说话 | 比台词更有力 |
| **细节特写** | 聚焦微小物件/动作 | 暗示、象征 |
| **对比** | 两个极端并置 | 荒诞、反思 |

---

## 六、对话技巧

| 手法 | 说明 |
|------|------|
| **话题回避** | 角色故意不聊真正该聊的 → 反而更明显 |
| **语言节奏** | 长短句交替，像音乐 |
| **潜台词对话** | 表面说A实际在说B |
| **沉默对峙** | 对话中断的沉默有时是最重要的台词 |
| **重复变奏** | 同一句话在不同场景说出，含义不同 |

---

## 手法组合速查

| 想要的效果 | 组合手法 |
|-----------|---------|
| 伤感但不煽情 | 留白 + 细节特写 + 冰山理论 |
| 紧张 | 交叉蒙太奇 + 限制视角 + 潜台词 |
| 意外反转 | 限制视角 + 不可靠叙述 + 反转 |
| 宿命感 | 预叙 + 环形结构 + 重复 |
| 荒诞 | 第三人称冷叙述 + 对比 + 降格 |
| 温暖治愈 | 慢镜头 + 动作即性格 + 细节特写 |
