# 剧本结构库

## 一、三幕结构（最通用）

```
第一幕：建制（Setup）
├── 展示日常世界
├── 激励事件（Inciting Incident）：打破平衡
└── 第一幕转折点：主角做出不可逆的选择

第二幕：对抗（Confrontation）
├── 上升行动：障碍递增
├── 中点转折：假胜利或假失败
├── 一败涂地（All Is Lost）：最低谷
└── 第二幕转折点：新的希望/方案出现

第三幕：解决（Resolution）
├── 高潮（Climax）：最终对决
├── 结局：新平衡建立
└── 终场画面：与开场画面对照
```

**适用**：大部分叙事作品，电影/动画/短视频均适用

## 二、Snyder 节拍表（15 拍）


| # | 节拍 | 页数占比 | 说明 |
|---|------|---------|------|
| 1 | 开场画面 | 1% | 故事开始前的世界 |
| 2 | 主题呈现 | 5% | 暗示核心主题 |
| 3 | 铺垫 | 1-10% | 主角的缺陷和缺失 |
| 4 | 催化剂 | 10% | 打破日常的事件 |
| 5 | 犹豫 | 10-15% | 主角纠结要不要行动 |
| 6 | 第一幕转折 | 20% | 进入新世界 |
| 7 | B故事 | 22% | 副线开始（常是爱情/友情） |
| 8 | 玩乐时光 | 20-50% | 新世界的乐趣和挑战 |
| 9 | 中点 | 55% | 假胜利或假失败 |
| 10 | 反派逼近 | 55-75% | 压力加大 |
| 11 | 一败涂地 | 75% | 最低谷 + "死亡气息" |
| 12 | 灵魂黑夜 | 75-85% | 最黑暗的时刻 |
| 13 | 第三幕转折 | 85% | 找到解决方案 |
| 14 | 结局 | 85-99% | 新世界的秩序 |
| 15 | 终场画面 | 100% | 与开场画面的对照 |

**适用**：商业电影、强情节故事

## 三、五阶段结构（古典叙事）

```
1. 开端 | 介绍人物、世界、初始状态
2. 发展 | 冲突升级，纠葛加深
3. 高潮 | 冲突到达顶点，命运决定
4. 下落 | 高潮余波，收束线条
5. 结局 | 新的平衡，主题回响
```

**适用**：古典戏剧、文学改编、史诗叙事

---

## 四、环形叙事

故事的开头和结尾形成闭环：

```
开头画面 ←→ 结尾画面（相似但含义不同）
├── 同一场景/台词/动作 —— 重复但意义已变
├── 中间展开非线性的探索
└── 回到起点时，角色已不同
```

**示例**：《肖申克的救赎》首尾的监狱大门、《千与千寻》首尾的隧道

**适用**：成长故事、身份探索

---

## 五、多线叙事

多条故事线并行，最终交汇：

```
线A ──────────────────┐
                       ├→ 交汇点
线B ──────────────────┤
                       │
线C ──────┐           │
          └───────────┘
```

**变体**：
- **同时同地**（《疯狂的石头》）
- **同时异地**（《云图》）
- **异时同地**（《百年孤独》）
- **异时异地但主题关联**（《通天塔》）

## 六、短片结构（5-10 场）

短片不求完整弧，求一击即中：

```
1. Hook（5秒内抓住注意力）
2. 日常状态的 1 个切片
3. 打破 → 反应（核心事件）
4. 情绪收束（反转/余韵/留白）
5. 终场 punch
```

**关键原则**：短片的结尾比开头重要 —— 不在于「发生了什么」，而在于「发生后你什么感觉」

---

## 七、短视频节奏模型（30-90秒）

```
0-3秒   | 钩子（Hook）
3-15秒  | 问题/冲突展示
15-25秒 | 情绪上升/转折
25秒-完 | 解决/反转/情绪释放
最后1秒 | 余韵/互动引导
```

---

## 结构选择指南

| 输入特征 | 推荐结构 |
|---------|---------|
| 有明确对手/对抗 | 三幕结构 |
| 成长/蜕变主题 | 环形叙事 |
| 群像/多条线索 | 多线叙事 |
| 一句话灵感/氛围为主 | 短片结构 |
| 短视频需求 | 短视频节奏模型 |
| 强情节/反转多 | Snyder 节拍表 |
