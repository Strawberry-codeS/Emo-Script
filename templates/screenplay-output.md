# 制作圣经输出模板

此文件定义完整制作圣经的输出格式。
下游工具（AIGC生成/剪辑/配音）按模块读取。

---

## 制作圣经结构

```
《片名》制作圣经
├── 一、世界观设定
├── 二、人物设定
├── 三、关键道具设定
├── 四、核心视觉装置设定（记忆墙/AI形态等）
├── 五、完整分镜头剧本
├── 六、全片节奏总谱
└── 七、AIGC制作优先级建议
```

---

## 一、世界观设定格式

```markdown
## 一、世界观设定

### 时间背景
[世界的时间/技术/社会背景，2-3句]
[这个世界里，技术和人的关系是什么]
[这个世界「缺少」什么]

### 主空间设定

**整体气质：**[2-3个关键词]

**色调：**[主色调K数] + [点缀色温说明]

**材质关键词：**
- 墙面：
- 地板：
- 家具：
- 窗：

**空间核心对比关系：**
[冷的区域：什么 → 情绪]
[暖的区域：什么 → 情绪]

**关键道具/装置：**[核心视觉元素的位置和功能]
```

---

## 二、人物设定格式

```markdown
## 人物设定

### 主角｜[名称/称谓]

**年龄外貌：**
**服装：**
**动作特征：**
**情感底色：**
**表演弧线：**[按阶段列出，几个层次，每层的外化动作]

### 次要角色｜[名称]
**出现方式：**[只出现什么，不出现什么]
**视觉处理：**[色调/透明度/边缘处理]
```

---

## 三、分镜头剧本格式

```markdown
### 镜头 XX｜时间区间

景别：[极近景/近景/中景/全景]

内容：
[场景描述，包含：谁在做什么，背景发生什么，
关键道具状态，观众此刻知道的 vs 角色知道的]

打光：
[色温] + [光源位置/方向] + [光质描述]
[特殊光效说明]

氛围：
[这一帧对观众传递什么情绪/信息]
[为什么这个镜头在情绪弧线上是必要的]

AIGC提示词方向：
[分点列出：主体描述/动作/环境/特殊效果/注意事项]
```

---

## 四、节奏总谱格式

```markdown
## 全片节奏总谱

时间轴       内容                色温        情绪          节奏
──────────────────────────────────────────────────────────
0–Xs        [内容]              [色温]       [情绪]        [快/中/慢/极慢]
Xs–Xs       [内容]              [色温]       [情绪]        [节奏]
...
```
